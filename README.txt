COMO ABRIR O SITE NO VISUAL STUDIO CODE

1. Baixe e descompacte a pasta armagedon-site.
2. Abra o Visual Studio Code.
3. Clique em File > Open Folder e selecione a pasta armagedon-site.
4. Instale a extensão Live Server no VS Code.
5. Clique com o botão direito no arquivo index.html.
6. Clique em Open with Live Server.

IMPORTANTE SOBRE O FORMULÁRIO

O arquivo contato.php só funciona em hospedagem com PHP configurado.
No Live Server, o visual do site funciona, mas o envio real do formulário pode não funcionar.
Enquanto o site estiver apenas em teste, o botão de WhatsApp já funciona normalmente.

ARQUIVOS PRINCIPAIS

index.html: estrutura e textos do site.
styles.css: cores, layout e aparência.
script.js: menu mobile e ano automático no rodapé.
contato.php: envio do formulário por e-mail em servidor com PHP.
assets: imagens usadas no site.
